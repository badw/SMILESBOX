#### Installation:

```
git clone https://github.com/badw/smilesbox.git 
cd smilesbox 
pip install . 
```

### Examples

an example Jupyter Notebook can be found here: 


`./examples/example.ipynb`


```example
from smilesbox.smilesbox import SMILESbox
sb = SMILESbox()
sb.smiles_to_atoms('CC')
sb.add_box([10,10,10])
sb.save('molecule.vasp')
```
